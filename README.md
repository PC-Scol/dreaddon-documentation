# dreaddon-documentation

Cet addon met à disposition de dremgr/web la documentation spécifique aux versions

Pour utiliser cet addon, rajouter ceci dans la configuration:
~~~sh
ADDON_URLS="
...
PC-Scol/dreaddon-documentation.git
...
"
~~~
